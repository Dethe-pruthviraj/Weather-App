# Weather App

## Project Description
A Python Weather Application built using Tkinter and the OpenWeatherMap API.

## Features
- Search weather by city
- Temperature
- Humidity
- Pressure
- Wind Speed
- Weather Description

## Technologies Used
- Python
- Tkinter
- Requests
- OpenWeatherMap API

## Installation

```bash
pip install -r requirements.txt
python main.py
```
